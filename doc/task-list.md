# 実装タスクリスト：観葉植物デスクトップアプリ

Claude Codeで上から順に実装する。

---

## Phase 1: プロジェクトセットアップ

- [ ] 1-1. Electron + React + TypeScript のプロジェクトを初期化する（`electron-vite`使用）
- [ ] 1-2. 依存パッケージをインストールする（`uiohook-napi`, `electron-store`）
- [ ] 1-3. メインプロセス・レンダラープロセスのディレクトリ構成を整える
- [ ] 1-4. 透過ウィンドウ・常駐設定をメインプロセスに実装する（透過・最前面・右下固定・クリック透過）
- [ ] 1-5. 開発環境でウィンドウが正しく表示されることを確認する

## Phase 2: 入力検知・ポイントエンジン

- [ ] 2-1. `uiohook-napi`でグローバル入力フックを実装する（キー打鍵・クリック・マウス移動）
- [ ] 2-2. 各入力を成長ポイントに換算するロジックを実装する
- [ ] 2-3. アイドル検出を実装する（一定時間入力なしでカウント一時停止）
- [ ] 2-4. `electron-store`で累積ポイント・成長段階をローカル保存する
- [ ] 2-5. アプリ再起動後に保存済み状態を復元することを確認する

## Phase 3: 成長システム

- [ ] 3-1. 成長フェーズの状態管理を実装する（苗・つぼみ・開花）
- [ ] 3-2. しきい値判定ロジックを実装する（50%でつぼみ・100%で開花）
- [ ] 3-3. 開花時の植物ランダム抽選ロジックを実装する
- [ ] 3-4. 開花後にポイント加算を一時停止する処理を実装する
- [ ] 3-5. 「次の苗を植える」でカウントリセット・苗に戻る処理を実装する

## Phase 4: UI実装

- [ ] 4-1. 植木鉢の固定表示コンポーネントを実装する
- [ ] 4-2. 成長段階に応じて植物画像を差し替える表示コンポーネントを実装する
- [ ] 4-3. 植物画像のプレースホルダー（仮画像）を用意する（苗・つぼみ・花 各1種）
- [ ] 4-4. 右クリックメニュー（コンテキストメニュー）を実装する
  - 「次の苗を植える」
  - 「図鑑を開く」
  - 「プライバシーについて」
  - 「終了」

## Phase 5: 図鑑機能

- [ ] 5-1. 図鑑データのスキーマを定義し`electron-store`で保存する（種類・初回開花日・累計開花回数）
- [ ] 5-2. 開花時に図鑑へ自動登録する処理を実装する
- [ ] 5-3. 図鑑閲覧画面のUIを実装する（リスト・詳細・コンプリート率）
- [ ] 5-4. 図鑑ウィンドウを別ウィンドウで開く処理を実装する

## Phase 6: プライバシー・初回起動

- [ ] 6-1. 初回起動時にプライバシー説明ダイアログを表示する
- [ ] 6-2. 同意済みフラグを保存し、2回目以降は表示しない処理を実装する
- [ ] 6-3. タスクトレイアイコンを実装する
- [ ] 6-4. タスクトレイメニューに「プライバシーについて」項目を追加する

## Phase 7: 調整・仕上げ

- [ ] 7-1. 植物画像を本番用に差し替える（種類を複数追加）
- [ ] 7-2. ポイント換算バランスを実使用で調整する
- [ ] 7-3. アプリアイコンを設定する
- [ ] 7-4. ビルド・パッケージング設定を整える（`electron-builder`）
- [ ] 7-5. Windows / Mac で動作確認する
